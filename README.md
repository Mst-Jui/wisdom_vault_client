# 📚 Wisdom Vault

A modern full-stack web application where users can create, organize, and share meaningful life lessons, personal experiences, and wisdom. The platform encourages self-growth through reflection while allowing users to explore lessons shared by the community.

# 🚀 Project Purpose

Wisdom Vault is designed to help people preserve valuable life lessons instead of forgetting them over time. Users can document personal experiences, organize them into categories, interact with community lessons, and unlock premium content through a one-time upgrade.

---

# ✨ Features

* 🔐 Secure authentication using Better Auth (Email/Password & Google Login)
* 👤 User & Admin Dashboard
* 📚 Create, Update & Delete Life Lessons
* 🌍 Public & Private Lesson Visibility
* ⭐ Free & Premium Lesson Access Control
* 💳 Stripe Payment Integration for Lifetime Premium Membership
* ❤️ Like & Unlike Lessons
* 🔖 Save Lessons to Favorites
* 💬 Comment System
* 🚩 Report Lesson Feature
* 🌟 Featured Lessons Controlled by Admin
* 🏆 Top Contributors Section
* 📖 Most Saved Lessons Section
* 📊 Dashboard Analytics with Weekly Activity Chart
* 👑 Role-based Access (User/Admin)
* 🌙 Dark & Light Theme Toggle
* 📱 Fully Responsive for Mobile, Tablet & Desktop
* 🔔 Beautiful Toast Notifications
* ⚡ Loading Spinner & Custom 404 Page

# ⚙️ Installation

Clone the repositories

```bash

```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
